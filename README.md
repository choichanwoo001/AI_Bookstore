## 🤔 개발 과정
첫 프로젝트이자 공부하면서 병행했던 실습? 이었다. 

원래는 쭉 개발을 완성하려고 했지만 여러가지 이유로 포기하게 되었다. 
1. 과에 중고책방이 필요할까?
2. 서버 운영비를 학생회비로 대체할 수 있는가?

이 두가지가 가장 큰 고민이자 해결해야할 문제였고 
결국은 해결책을 찾지 못한채 이 프로젝트는 끝나게 되었다.
